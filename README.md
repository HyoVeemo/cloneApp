## clone App

#

### 설치 및 실행

- git

```sh
$ git clone https://github.com/juntaekHong/cloneApp.git
```

- npm install

```
$ npm install
```

- react-native linking

```
$ react-native link // react-native version 0.6 이상이므로 link 필요없음
```

- app run

```
$ react-native run-android
$ react-native run-ios
```
